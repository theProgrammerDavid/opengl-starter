
# opengl-starter

Starter project for OpenGL in C++. Uses GLEW and GLFW

## Dependencies

* GLEW
````sudo apt-get install -y libglew-dev````

* GLFW
```sudo apt-get install -y libglfw3-dev```


## Instructions

* Source files (```.cpp```) go in ```src/```

* Header files go in ```include/```
